# AI Student Burnout Risk Predictor

## 📌 Project Overview
This project predicts student burnout risk based on study habits, sleep patterns, and screen time using machine learning.

## 🧠 Features
- Predicts burnout risk level
- Uses Logistic Regression model
- Includes data visualization
- Saves trained model for reuse

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Joblib

## 📊 Inputs
- Study Hours
- Sleep Hours
- Screen Time

## 🎯 Output
- Burnout Risk Prediction (High or Low)
